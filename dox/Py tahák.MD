# Python tahák

## Slicing

```python
a[start:stop]       # Od start do stop-1
a[start:]           # Od start do konce
a[:stop]            # Od  nuly do stop-1

a[-1]               # Poslední prvek
a[-2:]              # Poslední dva prvky (od start do konce) (-2 je předposlední prvek)
a[:-2]              # Vše kromě posledních dvou prvků (od nuly do stop-1)
```
