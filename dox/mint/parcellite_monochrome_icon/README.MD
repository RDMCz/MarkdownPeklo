# Parcellite monochrome tray icon

## EN

Source: [Ubuntu Mono Parcellite Icon](https://www.deviantart.com/ramzea/art/Ubuntu-Mono-Parcellite-Icon-161002817) by [Ramzea](https://www.deviantart.com/ramzea/gallery) under CC BY-SA 3.0 license.

I am redistributing the icons here, because you need to create an account to download them from DevianArt.

On Linux Mint 22 Cinnamon x64:

1. Download either `parcellite_monochrome_dark.svg` or `parcellite_monochrome_light.svg`.
2. Move the file to `/usr/share/pixmaps`, you will need root privileges for that.
3. In Parcellite _Preferences_, go to the _Display_ section.
4. Write the icon name into _Parcellite Icon Name_ input without the file extension.
   * Either `parcellite_monochrome_dark` or `parcellite_monochrome_light`
5. Click _OK_ and the icon should change immediately.

## CZ

Zdroj: [Ubuntu Mono Parcellite Icon](https://www.deviantart.com/ramzea/art/Ubuntu-Mono-Parcellite-Icon-161002817) od uživatele [Ramzea](https://www.deviantart.com/ramzea/gallery) dostupné pod licencí CC BY-SA 3.0.

Ikony redistribuuji zde, protože pro jejich stažení z webu DevianArt je potřeba zakládat účet.

Na Linux Mint 22 Cinnamon x64:

1. Stáhněte si `parcellite_monochrome_dark.svg` nebo `parcellite_monochrome_light.svg`.
2. Soubor přesuňte do složky `/usr/share/pixmaps`, tato akce vyžaduje oprávnění root.
3. V nastavení Parcellite (pravý klik na ikonku → _Preferences_), přejděte do sekce _Display_.
4. Název přesunutého souboru bez přípony napište do kolonky _Parcellite Icon Name_.
   * Buď `parcellite_monochrome_dark`, nebo `parcellite_monochrome_light`
5. Klikněte na _OK_ a ikona by se měla změnit.
